夏祭りの花火大会 を matplotlib で描く Python スクリプトを書いてください。

要件:
- 夜空に大きな打ち上げ花火が開いている夏祭りの情景を描く
- 花火は放射状の光の筋で2〜3発、色を変えて描く
- 画面下部に提灯の付いた屋台と人々のシルエットを置く
- 夜空は深い紺色のグラデーション風にして星を散らす
- 光の反射や提灯の灯りなど、夜らしい演出を入れる

実装ルール:
- matplotlib だけを使う
- 出力ファイルパスは sys.argv[1] から受け取る
- 画像は 1200x900 相当で保存する
- 軸や余計なラベルは出さない

出力ルール:
- 説明は不要
- Markdown の code fence も不要
- Python コード本体だけを出力する

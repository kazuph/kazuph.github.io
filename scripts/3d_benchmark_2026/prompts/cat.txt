かわいい猫のインタラクティブな 3D モデルを表示する HTML ページを作ってください。

要件:
- かわいい猫 1 匹を座りポーズで作る
- 頭、耳、マズル、目、ひげ、前脚、胴体、しっぽを部品として作り分ける
- しっぽがゆらゆら動くなど、生きている感じのアニメーションを 1 つ以上付ける
- 毛色・柄は自由。かわいさ重視で配色する

実装ルール:
- 単一の HTML ファイルだけで完結させる
- 3D 描画には Three.js を使ってよい。CDN（https://cdn.jsdelivr.net や https://unpkg.com）からの読み込みは可。それ以外の外部ライブラリは使わない
- 外部の画像・テクスチャ・3D モデルファイル（glTF など）の読み込みは禁止。形状はすべてコードで構築する
- マウスドラッグで回転、ホイールでズームできるようにする（OrbitControls 可）
- 画面全体に描画し、ウィンドウリサイズに追従する

出力ルール:
- 説明は不要
- Markdown の code fence も不要
- HTML 本体だけを出力する
